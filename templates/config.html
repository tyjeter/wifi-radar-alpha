Detection

Lower = more sensitive. Default: 2.0. Increase if too many false positives.
Sensitivity × this = fall threshold. Default: 3.0.

Alert Zones

Zones defined by RSSI range. An alert is fired when movement is detected within the zone's RSSI range (rough proximity indicator).

Home Assistant Integration

Create an automation in HA with a webhook trigger. Events: movement, fall, zone_alert, test.
Only needed if your HA webhook requires authentication.
